# Privacy Policy for SSM DRMGL

Last updated: 2026-06-24

SSM DRMGL is a personal hobby project developed by an individual developer.
This application does not collect, store, or transmit any personal information.

## 1. Data Collected by the App
SSM DRMGL processes the following data only on the user’s device:

- Vehicle ECU data (SSM2 protocol)
- USB device information (connected cable type)
- Camera preview frames (for HUD overlay)
- Video recordings created by the user
- CSV log files created by the user

All data is processed **locally on the device** and is **never uploaded** to any server.

## 2. How Data Is Used
The collected data is used only for the following purposes:

- Displaying real‑time ECU values
- Creating HUD overlays
- Saving video recordings to local storage
- Saving CSV logs to local storage

No data is used for analytics, tracking, advertising, or profiling.

## 3. Data Sharing
SSM DRMGL does **not** share any data with:

- Third‑party companies
- External servers
- Analytics services
- Advertising networks

All data stays on the user’s device unless the user manually exports it.

## 4. Permissions Used

### USB Access
Used to communicate with FTDI / OpenPort cables.
No USB data is transmitted outside the device.

### Camera Access
Used only for HUD preview and video recording.
No images or video are uploaded or shared.

### Microphone Access (optional)
Used only when recording video with audio.

### Storage Access
Used only to save videos and CSV logs created by the user.

The app does **not** access any other files on the device.

## 5. Children’s Privacy
This application is not intended for children.
No personal information is collected from any user.

## 6. Changes to This Policy
This policy may be updated in the future.
Updates will be reflected in this document.

## 7. Contact
For questions about this policy, please contact the developer via GitHub Issues:

https://github.com/maoh3/SSM_DRMGL_DOC/issues


# SSM DRMGL プライバシーポリシー

最終更新日: 2026-06-24

SSM DRMGL は個人開発の趣味プロジェクトです。
本アプリは個人情報を収集・保存・送信しません。

## 1. アプリが扱うデータ
SSM DRMGL は以下のデータを **端末内のみで処理**します：

- 車両 ECU データ（SSM2 プロトコル）
- USB デバイス情報（接続されたケーブル種別）
- カメラプレビュー映像（HUD 用）
- ユーザーが録画した動画
- ユーザーが保存した CSV ログ

これらのデータは **外部に送信されません**。

## 2. データの利用目的
データは以下の目的にのみ使用されます：

- ECU 値のリアルタイム表示
- HUD オーバーレイの描画
- 動画録画の保存
- CSV ログの保存

分析・広告・追跡などには一切使用しません。

## 3. データの共有
本アプリは以下とデータを共有しません：

- 第三者企業
- 外部サーバー
- 分析サービス
- 広告ネットワーク

データはすべて端末内に留まり、  
ユーザーが手動でエクスポートしない限り外部に送信されません。

## 4. 使用する権限

### USB アクセス
FTDI / OpenPort ケーブルとの通信に使用します。  
外部送信は行いません。

### カメラ
HUD 表示および動画録画のために使用します。  
画像や動画を外部に送信することはありません。

### マイク（任意）
動画録画時の音声収録に使用します。

### ストレージ
ユーザーが作成した動画・CSV ログの保存にのみ使用します。  
その他のファイルにはアクセスしません。

## 5. 子どものプライバシー
本アプリは子ども向けではありません。  
個人情報の収集は行いません。

## 6. ポリシーの変更
必要に応じて本ポリシーを更新する場合があります。  
更新は本ドキュメントに反映されます。
